# PHP

## 快捷鍵

- Ctrl ****+ \ 分割視窗
- Ctrl + / 單行註解
- Alt + Shift + A 多行註解
- Alt + Shift + 下 向下複製
- Alt + Shift + 上 向上複製
- Alt + 下 向下移動
- Alt + 上 向上移動

## 基礎

### 變數命名規則

1. 首字只可使用英文
2. 大小寫有別
3. 符號只可使用底線
4. 不可使用保留字

### 資料型態

* 字串(String)
* 整數(Int)
* 浮點數(Float)
* 布林(Boolean)
